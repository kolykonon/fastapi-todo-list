# Task Manager API 🚀

RESTful API для управления задачами с системой аутентификации пользователей. Проект построен на FastAPI с использованием Docker Compose для контейнеризации и Poetry для управления зависимостями.

## 📋 О проекте

Task Manager API - это backend-сервис для создания и управления задачами. Проект разработан в качестве пет-проекта для демонстрации навыков разработки на FastAPI, работы с асинхронностью, JWT-аутентификацией и современными инструментами разработки.

### ✨ Основные возможности

- ✅ Регистрация и аутентификация пользователей (JWT tokens)
- ✅ Управление задачами (CRUD операции)
- ✅ Фильтрация задач по статусу
- ✅ Защищенные эндпоинты с проверкой прав доступа
- ✅ Валидация данных через Pydantic schemas
- ✅ Асинхронная работа с базой данных
- ✅ Полная контейнеризация с Docker Compose
- ✅ Управление зависимостями через Poetry

## 🛠 Технологический стек

- **Python 3.13** - язык программирования
- **FastAPI** - современный веб-фреймворк
- **PostgreSQL 16** - база данных
- **SQLAlchemy** - ORM для работы с БД
- **Alembic** - миграции базы данных
- **Docker & Docker Compose** - контейнеризация и оркестрация
- **Poetry** - управление зависимостями и packaging
- **Pydantic V2** - валидация данных
- **pyjwt** - работа с JWT токенами
- **bcrypt** - хеширование паролей
- **python-multipart** - обработка form данных

## 🚀 Быстрый старт

### Предварительные требования

- Docker и Docker Compose (рекомендуемый способ)
- Python 3.13+ и Poetry (для локальной разработки)

### Запуск через Docker Compose

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/kolykonon/fastapi-todo-list.git
cd task-manager-api
```
2. **Настройте переменные окружения**
```bash
cp .env.example .env
```
3. **Сборка и запуск**
```bash
docker-compose up --build -d
```
4. **Проверка работы**
- **API**: <http://localhost:8000>
- **Swagger документация**: <http://localhost:8000/docs>
